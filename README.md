# About

This is a playground where I experiment with different rust technologies to
implement an [order book](https://en.wikipedia.org/wiki/Order_book).

# Technologies

* [Tokio](https://tokio.rs/)
* [Axum](https://docs.rs/axum/latest/axum/)
* [sqlx](https://docs.rs/sqlx/latest/sqlx/)
* [Cucumber](https://docs.rs/cucumber/latest/cucumber/)
* [pgroll](https://github.com/xataio/pgroll)
* [Terragrunt / Opentofu](https://terragrunt.com/)
* [Tracing](https://docs.rs/tracing/latest/tracing/)
